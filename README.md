# Автотесты webcalculator

## О проекте
Приложение webcalculator.exe — это HTTP-сервер который принимает 
арифметические задачи через API и возвращает результат вычисления 
в формате JSON. Данный репозиторий содержит полный набор автоматических 
тестов для проверки его функциональности, написанных на Python 
с использованием pytest.

## Структура тестов
Тесты разбиты на четыре логических блока, каждый из которых 
соответствует требованиям задания.

**TestResponse** — проверяет что сервер возвращает данные в правильной 
структуре: наличие нужных ключей в JSON, правильные типы данных, 
поддержку методов GET/POST/OPTIONS.

**TestArithmetic** — проверяет корректность всех четырёх арифметических 
операций (сложение, умножение, целочисленное деление, остаток от деления) 
на различных входных данных: положительные и отрицательные числа, 
операции с нулём, граничные значения int32.

**TestManagement** — проверяет команды start, stop, restart через 
командную строку, значения хоста и порта по умолчанию, запуск 
на нестандартном порту, команду show_log и флаг --help.

**TestNegative** — проверяет корректность кодов ошибок при неправильных 
входных данных: отсутствующие ключи, строки вместо чисел, числа 
с плавающей точкой, превышение границ int32, деление на ноль, 
невалидный JSON.

## Как запустить тесты

Шаг 1. Убедитесь что webcalculator.exe и test_webcalculator.py 
находятся в одной папке.

Шаг 2. Установите зависимости:
pip install -r requirements.txt

Шаг 3. Запустите тесты с генерацией отчёта:
pytest test_webcalculator.py -v --html=report.html --self-contained-html

После выполнения в папке появится файл report.html с подробным 
отчётом по каждому тесту.

## Обнаруженные дефекты в приложении
В ходе тестирования были выявлены три дефекта.

1. Деление на ноль — при делении на 0 приложение не возвращает 
код ошибки 1, как указано в спецификации.

2. Остаток от деления на ноль — вместо кода ошибки 1 приложение 
возвращает число 8, что математически бессмысленно.

3. Превышение границ int32 — при передаче числа больше 2147483647 
приложение не возвращает код ошибки 4, как указано в спецификации.
